# Toolboard for MacOS

## Features

- simple window manager
- caffeinate

## Config file

Configuration file must be placed at one of the following places (in order):
  
- `$HOME/.config/toolboard/config.yml`
- `$HOME/.toolboard.yml`

Sample:

```yml
  window_padding: 10.0
  window_gap: 10.0
  border_width: 2.0
  border_radius: 10.0
  accent_color: "#44fcee"
```
